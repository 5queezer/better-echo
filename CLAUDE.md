# CLAUDE.md

## Project

better-echo — real-time speech-to-text with LLM-powered transcription correction. Python 3.13, uses `uv` for dependency management.

## Commands

- `uv sync` — install dependencies
- `uv run python main.py` — run the app
- `make serve` / `make serve-diart` — convenience targets

## Changelog

**Every pull request must update the Changelog section in README.md.** Add a bullet under today's date heading (create the heading if it doesn't exist yet). Format:

```markdown
### YYYY-MM-DD
- Short description of the change (#PR_NUMBER)
```

Keep entries in reverse chronological order (newest date first). Each bullet should be a concise, user-facing summary of what changed.
